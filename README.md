# BTC Gift Cards

## Features

- Deposit or mint ckBTC
- Create gift cards
  - Assign an amount of ckBTC to be send with it
  - Specify recipient
  - Add custom message


## local setup

1. create canisters
2. deploy ledger (./external/setup_ledger.sh)
3. dfx deploy
